# Personal Portfolio

This is a personal portfolio website built using standard web technologies. It serves as a digital resume to showcase projects, skills, and services offered.

## 📂 Project Structure

- `index.html` - The home page and main entry point.
- `about.html` - Information about my background and experience.
- `services.html` - Details about the services I provide.
- `contact.html` - A contact form and ways to reach me.
- `css/style.css` - The stylesheet containing all the styling and layout rules.
- `js/main.js` - The JavaScript file for interactive functionality.

## 🛠️ Technologies Used

- **HTML5:** For structuring the page content.
- **CSS3:** For styling, layout, and responsiveness.
- **JavaScript (Vanilla):** For adding interactivity to the website.

## 🚀 How to Run

Since this is a static website, running it is very simple:

1. Clone this repository or download the source code files.
2. Navigate to the project folder.
3. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge).

Alternatively, you can use the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension in VS Code for real-time reloading during development.
